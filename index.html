Features

Work in progress
Intuitive

Theming
Custom properties allow for adoption into any design system.
Accessible
Designed to be usable by anyone.
Fast
Optimized for bundle size and runtime performance.
Runs everywhere
Framework and library agnostic custom elements.
Keyboard navigation
Fully controllable by keyboard.

Beta

  • All behavior covered by E2E and unit tests.
  • Documentation on this showcase.
Stable
  • IDE "IntelliSense" Support.
  • Angular facade.
  • Accessibility.
  • Pre-built themes.
